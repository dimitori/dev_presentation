Единое древо директорий

В Linux (unix) более прострая иерархия файлов

  • / - корень
  • /etc - конфигурация
  • /home - данные пользователей
  • /usr - приложения
  • /var - данные и логи
  • /tmp - временные файлы

Терминал - это командная строка
